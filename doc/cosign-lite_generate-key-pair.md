## cosign-lite generate-key-pair

Generate a local password-encrypted key pair

```
cosign-lite generate-key-pair [flags]
```

### Options

```
  -h, --help                       help for generate-key-pair
      --output-key-prefix string   name used for generated .pub and .key files (default "cosign")
```
